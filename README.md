    Colégio Estadual Professora Cleia Godoy Fabrini Silva, para o 
    concurso agrinho 2026 promovido pelo Sistema FAEP/SENAR-PR em 
    parceria com o Governo do Estado do Paraná

## Descrição

O projeto Agrinho é uma iniciativa voltada para educação e tecnologia no meio rural. Este README apresenta informações básicas sobre o conteúdo e objetivo do material disponível.

## Conteúdo

- Documentação do projeto
- Recursos e arquivos de distribuição
- Orientações de uso e instalação

## Como usar

1. Abra a pasta `EcoAgro-Monitor`.
2. Verifique os arquivos presentes.
3. use um live preview no arquivo layout.html ou acesse via vercel no seu navegador usando o Link: [Agroteen](https://agroteen.vercel.app).
