# Projeto Agrinho

Este repositório contém os arquivos finais do projeto Agrinho na pasta `.dist`.

## Descrição

O projeto Agrinho é uma iniciativa voltada para educação e tecnologia no meio rural. Este README apresenta informações básicas sobre o conteúdo e objetivo do material disponível.

## Conteúdo

- Documentação do projeto
- Recursos e arquivos de distribuição
- Orientações de uso e instalação

## Como usar

1. Abra a pasta `EcoAgro-Monitor`.
2. Verifique os arquivos presentes.
3. use um live preview no arquivo layout.html ou acesse via vercel no seu navegador usando o link:.

## Observação

Este projeto e apenas um demonstração, alguns dos dados pode ser inventados ou ainda não completados.
